Instructions

  1. Install the Elementor Copier Chrome extension
  2. Right-click on any widget or section above
  3. Select "Copy Widget" or "Copy Section" from the context menu
  4. Check the browser console for media extraction logs
  5. Paste the clipboard data to verify the media array is populated

Expected Results:

  • Test 1: Should extract 1 regular image
  • Test 2: Should extract 1 background image from CSS
  • Test 3: Should extract 1 background image from inline style
  • Test 4: Should extract 1 video source
  • Test 5: Should extract 1 YouTube iframe URL
  • Test 6: Should extract 4 images (1 src + 3 srcset)
  • Test 7: Should extract 3 images (1 section background + 2 column images)
  • Test 8: Should extract 1 lazy-loaded image
