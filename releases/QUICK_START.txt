QUICK START - CHROME WEB STORE SUBMISSION
================================================================================

1. GO TO CHROME WEB STORE DEVELOPER DASHBOARD
   https://chrome.google.com/webstore/devconsole

2. CLICK "NEW ITEM"

3. UPLOAD FILE
   releases/elementor-copier-v1.0.0.zip

4. FILL STORE LISTING
   - Name: Elementor Copier
   - Category: Developer Tools
   - Description: See CHROME_WEB_STORE_SUBMISSION.md

5. UPLOAD SCREENSHOTS
   - At least 1 screenshot (1280x800 or 640x400)
   - Show extension in action

6. JUSTIFY PERMISSIONS
   - See CHROME_WEB_STORE_SUBMISSION.md for justifications

7. SUBMIT FOR REVIEW
   - Review time: 1-3 business days

TESTING BEFORE SUBMISSION
================================================================================

1. Extract the ZIP file
2. Open Chrome -> chrome://extensions/
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select extracted folder
6. Test all features:
   - Elementor detection
   - Context menu
   - Highlight mode
   - Copy operations
   - Clipboard data
