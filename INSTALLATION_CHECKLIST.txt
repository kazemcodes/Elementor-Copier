⚠️ IMPORTANT REMINDERS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Elementor Required:
   This plugin REQUIRES Elementor to be installed and activated.
   Install Elementor first if you haven't already.

2. Current Limitation:
   v1.0.0 extracts content but doesn't import yet.
   Import functionality coming in v1.1.0 (Task 20).

3. Security:
   • Use Application Passwords (most secure)
   • Use HTTPS on both sites
   • Don't share credentials
   • Revoke access when done

4. Performance:
   • Start with small batches (5-10 items)
   • Increase batch size gradually
   • Monitor server resources
   • Use during off-peak hours

5. Backup:
   Always backup your site before major operations.
